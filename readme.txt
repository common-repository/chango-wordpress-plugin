Easily integrate Chango PPC search targeted ad units into your Wordpress Blog

== Description ==
Easily integrate Chango PPC search targeted ad units into your Wordpress Blog.  Earn money from your blogs SEO optimized traffic. To see a demo of Chango's ad units, navigate to: http://www.chango.com/publishers/demo

== Installation ==
1. Download the plugin
2. Upload the Chango plugin files to the `/wp-content/plugins/chango/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Go to the Chango settings page and customize your options.   You will need to paste the ad unit text into the settings area.  Login to Chango at https://www.chango.com/login to choose your ad unit size and to copy the javascript.
5. That's it!

If you have to upgrade manually simply repeat the installation steps and re-enable the plugin.

== Configuration options ==
* Insert ads at top or bottom of posts, or x paragraphs from top or bottom of post
* Turn off/on all ads on blog mainpage
